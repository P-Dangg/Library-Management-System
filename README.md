# Library-Management-System
A Java-based Library Management System for librarians to manage patron records, add/remove patrons, and import data from text files.

# TARGET USERS:
Librarians and library staff who need a simple, reliable way to 
manage patron information without the complexity of a full database 
system.

# KEY FEATURES:
- Add patrons manually through the command line
- Import multiple patrons from a text file
- Remove patrons using their unique ID
- Display a complete list of all patrons
- Search for patrons by ID
- Validate all input data to ensure accuracy
